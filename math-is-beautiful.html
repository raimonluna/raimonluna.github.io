Math is Beautiful

A gallery of generative mathematics

Square images generated from code — fractals, dynamical systems, and geometric constructions. Click any tile for the full image, the story behind it, and a link to the code that made it.

@raimon.luna on Instagram →

To add a new piece: duplicate one of the <button class="mb-tile"> blocks above with new files and new data-* attributes. Use the first pattern for static images, the second for short animations — whether the lightbox shows an image or plays a video is decided automatically from the file extension in data-full (.mp4 / .webm → video). For videos, keep a still "poster" frame as the grid thumbnail (data-thumb) since the grid never autoplays — only the lightbox does.
