Math is Beautiful

A gallery of mathematical visualizations

I started this project as an Instagram channel for visualizations of mathematical concepts generated from simple Python scripts. Click any tile for the full image, the story behind it, and a link to the code that made it. Unlike the backbone HTML/CSS/JS code of this website, no AI has been used to write the Python code that generates these images, as I use it for my own logical and mathematical training. This collection was featured in a post in microsiervos.com.
